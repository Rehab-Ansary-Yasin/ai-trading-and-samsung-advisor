# AI Trading and Samsung Phone Advisor

This repository contains two independent projects developed as part of an AI and backend engineering assignment.

Both projects are runnable and demonstrated using Google Colab.

---

## Task 1: Algorithmic Trading Adventure

A Python-based algorithmic trading system using a class-based design.

### Key Features
- Uses historical stock data from `yfinance`
- Implements a Golden Cross strategy (50-day & 200-day moving averages)
- Buys on bullish crossover and sells on reversal
- Investment budget limited to $5000
- Automatically closes any open position on the final trading day
- Calculates final profit or loss
